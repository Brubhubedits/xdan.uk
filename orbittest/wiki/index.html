OrbitOS Documentation

Welcome to the OrbitOS wiki. OrbitOS is a Fedora-based Linux distribution designed for gaming and desktop use, featuring the latest KDE Plasma desktop environment and comprehensive gaming support out of the box.

OrbitOS was created with the goal of providing a seamless gaming experience on Linux while maintaining the stability and security that Fedora is known for. Our distribution comes with pre-configured gaming tools and optimizations, making it perfect for both Linux newcomers and experienced users.

Key Features

  • Based on Fedora 41 with rolling updates
  • KDE Plasma desktop environment with custom OrbitOS theme
  • Pre-configured gaming setup including Steam, Proton, and Lutris
  • Optimized performance for both AMD and NVIDIA GPUs
  • Extensive software repository access through RPM Fusion
  • Custom kernel optimizations for gaming performance
  • Enhanced system responsiveness with custom I/O scheduler
  • Simplified driver management
  • Modern package management with DNF

System Requirements

Component Minimum Recommended
Processor Dual-core x86_64 Quad-core x86_64
Memory 4GB RAM 8GB RAM or more
Storage 20GB 50GB or more
Graphics Any modern GPU NVIDIA GTX 1060 / AMD RX 570 or better

Installation Overview

Before proceeding with the installation guide, ensure you have the necessary system requirements and a compatible computer. The installation process is straightforward and should not take long.

Preparation

  • Backup your data if necessary
  • Download the OrbitOS ISO file from the official website
  • Create a bootable USB drive using the downloaded ISO

Installation Guide

Download Options

  • AMD/Intel ISO: orbit-amdgpu-latest.iso
  • NVIDIA ISO: orbit-nvidia-latest.iso

Installation Media Creation

  1. Download Rufus, Balena Etcher, or similar USB writing tool
  2. Insert a USB drive (minimum 8GB)
  3. Select the OrbitOS ISO in your writing tool
  4. Write to USB and wait for completion

BIOS/UEFI Setup

Before installation, you'll need to:

  1. Enter BIOS/UEFI (usually F2, F12, or Del during startup)
  2. Disable Secure Boot
  3. Set boot order to prioritize USB
  4. Save changes and exit

Installation Type Selection

Clean Install

Recommended for dedicated gaming machines:

  • Formats entire drive
  • Maximum performance
  • No complications from existing OS

Dual Boot

For users who need Windows access:

  • Requires existing free space or partition
  • Minimum 50GB recommended for OrbitOS partition
  • Windows should be installed first

Step-by-Step Installation

  1. Boot from USB and select "Install OrbitOS"
  2. Choose language and keyboard layout
  3. Select installation type:
    • For clean install: "Erase disk and install OrbitOS"
    • For dual boot: "Install alongside existing OS"
  4. If dual booting:
    • Use the slider to allocate space between systems
    • Ensure at least 50GB for OrbitOS
    • Consider your Windows partition size needs
  5. Set your timezone
  6. Create user account:
    • Choose username (lowercase, no spaces)
    • Set strong password
    • Option to enable automatic login (not recommended)
  7. Review settings and begin installation
  8. Wait for installation to complete (~15-20 minutes)
  9. Restart when prompted
  10. Remove USB drive

Post-Installation

After first boot:

  • Complete initial setup wizard
  • Connect to internet
  • Run system updates
  • Install any additional drivers if needed
# Update system after installation
sudo dnf update

First Steps

After installing OrbitOS, you'll be greeted with a login screen. Enter your username and password to log in.

Initial Setup

  • Update your system: sudo dnf update
  • Explore the desktop environment and get familiar with the interface
  • Configure your network settings if necessary

KDE Plasma Configuration

OrbitOS ships with a customized KDE Plasma desktop. Here's how to customize it further:

Common Customizations

  • Right-click desktop → Configure Desktop
  • System Settings → Appearance
  • System Settings → Window Management

Changing Desktop Environment

OrbitOS is designed to work with KDE Plasma, but you can change to other desktop environments if you prefer.

Available Desktop Environments

  • GNOME
  • XFCE
  • Cinnamon

To install an alternative desktop environment, use the following command: sudo dnf groupinstall "Desktop Environment"

NVIDIA Setup

OrbitOS includes the NVIDIA drivers by default. If you need to reinstall or update them:

# Reinstall NVIDIA drivers
sudo dnf reinstall kernel-modules-nvidia

NVIDIA Configuration

You can configure NVIDIA settings using the NVIDIA X Server Settings application.

AMD Setup

OrbitOS includes the AMD drivers by default. If you need to reinstall or update them:

# Reinstall AMD drivers
sudo dnf reinstall kernel-modules-amdgpu

AMD Configuration

You can configure AMD settings using the AMD Control Center application.

Intel Setup

OrbitOS includes the Intel drivers by default. If you need to reinstall or update them:

# Reinstall Intel drivers
sudo dnf reinstall kernel-modules-i915

Intel Configuration

You can configure Intel settings using the Intel Graphics Control Panel application.

Steam Setup

Steam comes pre-installed with OrbitOS, ready to use out of the box. However, if you need to reinstall it for any reason:

# Reinstall Steam if needed
sudo dnf reinstall steam

Initial Configuration

OrbitOS comes with Steam pre-configured for optimal gaming performance. The following features are enabled by default:

  • Steam Play/Proton enabled for all titles
  • GameMode integration
  • MangoHud support
  • FSR (FidelityFX Super Resolution) support

Proton Guide

Proton is Valve's compatibility layer that allows Windows games to run on Linux. OrbitOS includes the latest stable version of Proton.

Using Different Proton Versions

To change Proton versions for a specific game:

  1. Right-click the game in your Steam library
  2. Select Properties
  3. Click Compatibility
  4. Check "Force the use of a specific Steam Play compatibility tool"
  5. Select your desired Proton version

GE-Proton Guide

GE-Proton (Glorious Eggroll) is a custom Proton build with additional patches and improvements.

Installing GE-Proton

OrbitOS includes ProtonUp-Qt for easy GE-Proton installation:

  1. Open ProtonUp-Qt from the application menu
  2. Click "Add version"
  3. Select "GE-Proton" from the compatibility tool dropdown
  4. Choose the latest version
  5. Click "Install"

Lutris Guide

Lutris is a game manager that supports various gaming platforms and emulators. It comes pre-installed with OrbitOS.

Basic Usage

  • Click the "+" button to add games
  • Search for games in the Lutris database
  • Use install scripts for automated setup

Running Non-Steam Games

Lutris provides easy installation for:

  • Epic Games Store
  • GOG Galaxy
  • Battle.net
  • Origin
  • Ubisoft Connect
